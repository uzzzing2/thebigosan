## 🔝 공통 레이아웃

### D-Day 띠 (TopBar)

**위치**: 모든 페이지 최상단

**텍스트**:
```
🗓️ 본선거 6.3까지 D-15  ·  사전투표 5.29~5.30
```

**사양**:
- 배경: `bg-red-500` (#EA2C38)
- 텍스트: `text-white` 14px Medium
- 패딩: 10px 16px
- 중앙 정렬
- 닫기 버튼 없음 (항상 노출)
- D-Day는 클라이언트 자동 계산 (선거일: `2026-06-03`)
- 선거 당일: "🗳️ 오늘은 본선거일입니다! 투표소 찾기"
- 모바일: 텍스트 줄임 ("D-15 · 사전 5.29~5.30")

### 헤더 (Header)

**위치**: D-Day 띠 아래, sticky

**구조**:
```
┌─────────────────────────────────────────────────┐
│  [캠프 로고]                  [메뉴 5개]         │
└─────────────────────────────────────────────────┘
```

**사양**:
- 높이: 72px (모바일 60px)
- 배경: white (스크롤 시 옅은 그림자)
- 컨테이너: max-w-container, 좌우 패딩
- 로고: 좌측, 클릭 시 홈
- 메뉴: 우측
  - 후보자 소개 / 성과 / 공약 / 보도자료 / 응원한마디
  - 16px Medium gray-900
  - 활성 메뉴: red-500 + 밑줄 (2px)
  - 호버: red-500
- 모바일: 햄버거 메뉴 (우측), 클릭 시 풀스크린 메뉴

**모바일 메뉴**:
- 풀스크린 오버레이
- 메뉴 항목 세로 나열, 22px Bold
- 닫기 X 버튼 우측 상단
- ESC 키 닫기

### 푸터 (Footer)

**배경**: `bg-cream-100`
**패딩**: py-16, px-base

**구조 (데스크톱 4분할, 모바일 세로 적층)**:


### 응원 리스트

배경: `bg-white`

```
[리스트 - 1열, 카드 사이 16px]
[응원 카드 20개]

[더 보기 버튼 중앙]

```

```
│                                       │
│ 더 큰 오산을 위해 함께하겠습니다!     │
│ 화이팅!                               │
│                                       │
│                              🚩 신고  │
└──────────────────────────────────────┘
```

카드 사양:
- 배경: white
- 모서리: radius-2xl
- 그림자: shadow-md
- 패딩: 24px
- 신고 버튼: 우측 하단, gray-500, 호버 시 red-500

## 🔐 관리자 페이지 (`/admin`)

### 로그인

Firebase Auth 이메일·비번. 환경 변수의 `ADMIN_EMAILS`에 등록된 이메일만 접근.

### 대시보드

```
[좌측 사이드바]                     [우측 콘텐츠]
- 대시보드                          
- 응원 관리                         [통계 카드 3개]
- 보도자료 관리                     - 누적 응원: 1,234
- SNS 큐레이션                      - 신고 대기: 5
- 카드 관리                         - 이번주 방문: 8,432
- 설정                              
                                    [최근 응원 5개]
                                    [최근 신고 5개]
```

### 응원 관리

- 전체 리스트 / 신고된 응원 탭
- 각 응원: 보기·삭제·닉네임 차단·IP 차단

### 보도자료 관리

- 새 글 작성 (Tiptap 에디터)
  - 카테고리, 제목, 본문, 썸네일, 날짜
  - 보도 언론 링크 (여러 개 추가)
- 수정·삭제·공개 토글

### SNS 큐레이션 (인스타)

- 게시물 추가: 이미지 URL, 캡션, 원본 URL, 날짜
- 순서 변경 (드래그)
- 활성/비활성 토글

### 카드 관리

- 성과 30개·공약 18개 관리
- 이미지 업로드, 중요도 순번, 메인 노출 토글

### 설정

- 금지어 사전 관리
- 운영자 추가/삭제

## 🎬 인터랙션 모션 가이드

| 요소 | 동작 |
|---|---|
| 페이지 진입 | 0.4s fade + translateY(8px) |
| 섹션 viewport 진입 | 한 번만 stagger 진입 |
| 버튼 호버 | 0.2s translateY(-2px) + 그림자 강화 |
| 카드 호버 | 0.3s translateY(-4px) + 그림자 강화 |
| 모달 등장 | 0.3s scale 0.96→1 + opacity + translateY 16→0 |
| 모달 퇴장 | 0.2s 빠르게 |
| 토스트 | 우측 상단(데스크톱)·하단(모바일), 3초 자동 |
| 카운터 숫자 | 페이지 진입 시 0 → 실제값 카운트업 (1.5s) |
| Marquee | 30s linear infinite, 호버 시 일시정지 |
